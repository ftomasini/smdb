Dados de conexão estão em config.php. As configurações relacionadas aos padrões e preferências estão nos arquivos config_pt_br.php e config_en_us.php

Cada um dos arquivos contém preferências específicas que são carredas conforme o idioma do usuário.

Lembre-se de manter os dois arquivos com as devidas configurações (ao alterar um, certifique-se em manter o outro equivalente).
